"# Kaspa Exchange Data Repository\n\nThis repository stores historical exchange data for KRC20 tokens using GitRows API integration.\n\n## Overview\n\nThis repository contains:\n- **Raw ticker data**: Minute-by-minute exchange data for all configured tokens\n- **Daily summaries**: OHLC (Open, High, Low, Close) aggregations per exchange\n- **EOD summaries**: End-of-day summaries aggregated across all exchanges\n\n## Repository Structure\n\n```\nKaspa-Exchange-Data/\n├── data/\n│   ├── {token}/\n│   │   ├── {exchange}/\n│   │   │   ├── {year}/\n│   │   │   │   ├── {month}/\n│   │   │   │   │   ├── {date}.json          # Daily OHLC summary\n│   │   │   │   │   └── {date}-raw.json     # Raw minute-by-minute data\n│   │   │   │   └── ...\n│   │   │   └── ...\n│   │   └── ...\n│   └── ...\n└── summaries/\n    └── daily/\n        ├── {token}/\n        │   ├── {date}-summary.json          # EOD summary\n        │   └── latest.json                  # Latest summary\n        └── ...\n```\n\n## Data Format\n\n### Raw Data File\n```json\n{\n  \"timeframe\": \"24h\",\n  \"token\": \"kaspa\",\n  \"exchange\": \"ascendex\",\n  \"resolution\": \"1m\",\n  \"count\": 132,\n  \"data\": [\n    {\n      \"timestamp\": 1766860574,\n      \"last\": 0.04477,\n      \"bid\": 0.0445,\n      \"ask\": 0.04484,\n      \"high\": 0.04512,\n      \"low\": 0.04214,\n      \"volume\": null,\n      \"quoteVolume\": 53620.27,\n      \"change\": 0,\n      \"percentage\": 0\n    }\n  ]\n}\n```\n\n### Daily Summary File\n```json\n{\n  \"date\": \"2025-12-28\",\n  \"token\": \"kaspa\",\n  \"exchange\": \"ascendex\",\n  \"open\": 0.04477,\n  \"high\": 0.04512,\n  \"low\": 0.04214,\n  \"close\": 0.04475,\n  \"volume\": 1234567.89,\n  \"quoteVolume\": 55123.45,\n  \"change\": -0.00002,\n  \"percentage\": -0.0447,\n  \"trades\": 1320,\n  \"avgPrice\": 0.04485,\n  \"vwap\": 0.04482\n}\n```\n\n## Accessing Data\n\n### Via GitRows API\n```\nhttps://api.gitrows.com/@github/KaspaDev/Kaspa-Exchange-Data/data/kaspa/ascendex/2025/12/2025-12-28.json\n```\n\n### Via Frontend API\n```\nGET /api/history/kaspa/ascendex?timeframe=24h\nGET /api/history/kaspa/ascendex?timeframe=7d\nGET /api/history/kaspa/ascendex?timeframe=30d\n```\n\n## Supported Tokens\n\n- **Kaspa** (KAS) - 119+ exchanges\n- **TBDai** - Multiple exchanges\n- **MATRIX** - GroveX, FameEx\n- **Nacho** - Multiple exchanges\n- **Zeal** - MEXC, CoinEx\n- And many more KRC20 tokens...\n\n## Data Updates\n\n- **Continuous Updates**: Data is committed every 5 minutes\n- **Daily Summaries**: Generated at end of day (EOD)\n- **Git Tags**: Created daily with format `eod-YYYY-MM-DD`\n\n## Version Control\n\nAll data is version-controlled in Git:\n- Full commit
